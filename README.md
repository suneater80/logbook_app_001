SRP sangat membantu ketika pengembangan software karena memiliki beberapa keunggulan
1. Kode menjadi lebih mudah dibaca karena lebih sederhana dan tidak berantakan
2. debugging lebih cepat, ketika ada yang error akan lebih cepat menemukan errornya
3. lebih mudah dimodifikasi, karena kode sudah dikelompokkan dengan posisi dan kegunaannya
